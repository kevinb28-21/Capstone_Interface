ğŸ†˜ Troubleshooting

If git pull fails:
Fetch and reset
git fetch origin main
git reset --hard origin/main

Or check git status
git status

If PM2 restart fails:
Check what's running
pm2 list

Stop all and restart
pm2 delete all
cd server
pm2 start ecosystem.config.cjs  # or ecosystem.config.js
pm2 save

If backend doesn't respond:
Check logs for errors
pm2 logs --lines 100

Check if port is in use
sudo netstat -tlnp | grep 5050

Check .env file
cd server
cat .env

If Instance Connect doesn't work:
  Make sure instance is "Running" (green circle)
  Try refreshing the AWS Console page
  Try a different browser
  Check if you have permissions to connect
