DOCUMENTATION INDEX
===================

This directory contains all project documentation organized by category.
Documentation is available in both Markdown (.md) and plain text (.txt) formats.

STRUCTURE
---------

documentation/
  frontend/          - Frontend-specific documentation (currently empty)
  backend/           - Backend integration and setup documentation
  ml/                - Machine learning, model training, and Python processing
  deployment/        - Deployment guides for EC2, Netlify, S3, and troubleshooting
  misc/              - Miscellaneous reports, audits, and project summaries
  api/               - API endpoint documentation (Flask and Node.js)
  database/          - Database schema and migration guides
  architecture/      - System architecture and design documentation
  getting-started/   - Quick start guides and project overview
  development/       - Local development setup and testing guides

FILE FORMATS
------------

Each documentation file is available in two formats:
  - .md files - Markdown format (for GitHub rendering and editing)
  - .txt files - Plain text format (for universal readability)

Both formats contain the same content, so you can use whichever is more convenient.
